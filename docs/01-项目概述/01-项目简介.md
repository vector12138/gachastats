# 项目简介

## 🎮 GachaStats - 米哈游抽卡数据分析工具

### 项目定位
GachaStats 是一个专注于米哈游全系列游戏的抽卡数据分析工具，旨在帮助玩家科学抽卡、合理规划资源。

### 核心价值
- **轻量易用**：无需复杂配置，开箱即用
- **完全本地化**：所有数据存储在本地服务器，保护隐私安全
- **多游戏支持**：支持米哈游旗下所有热门游戏
- **专业分析**：提供详细的抽卡数据统计和分析

### 适用人群
- 米哈游游戏玩家（原神、绝区零、星穹铁道、崩坏3）
- 需要管理多个游戏账号的玩家
- 希望科学规划抽卡资源的玩家
- 对抽卡数据分析感兴趣的用户

### 技术特点
- **前后端分离**：FastAPI后端 + 纯HTML前端
- **数据库**：SQLite轻量级数据库
- **部署简单**：单文件部署，无需复杂环境
- **跨平台**：支持Windows、Linux、macOS

### 开发理念
1. **用户友好**：界面简洁，操作直观
2. **数据安全**：所有数据本地存储，不上传云端
3. **功能专注**：专注于抽卡数据分析，不做冗余功能
4. **持续更新**：根据用户反馈持续优化和改进
