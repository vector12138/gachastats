# 🎮 GachaStats - 米哈游抽卡数据分析工具
专注于米哈游全系列游戏的抽卡数据分析工具，轻量、易用、完全本地化，帮你科学抽卡不做冤大头！
## ✨ 核心功能
### 📱 多账号多游戏支持
- ✅ 原神 / 绝区零 / 星穹铁道 / 崩坏3 全支持
- ✅ 任意数量账号统一管理，不用来回切换
### 📥 多种导入方式
- 🔗 **官方抽卡链接一键导入**：复制游戏内抽卡记录链接，一键同步全量历史数据，不用手动录入
- 📸 **截图OCR识别**：手机截图上传自动识别抽卡记录（可选功能）
- ✍️ **手动录入**：支持手动添加抽卡记录
### 📊 专业数据分析
- 🎰 **基础统计**：总抽数、五星数量、出货率、四星数量
- 📈 **卡池水位计算**：各个卡池当前抽数、下次出货概率、平均出货水位
- 🏆 **欧非评级**：根据抽卡情况给你评级，看看你是欧皇还是非酋
- 📜 **历史记录**：所有五星出货历史，支持查看每个角色的出货水位
- 📊 **多账号对比**：对比所有账号的抽卡情况，看看哪个号更欧
### 🎨 美观界面
- 📱 响应式设计，手机电脑都能用
- 🎨 渐变主题，好看又好用
- 🌐 Web界面，直接用浏览器访问，不需要安装客户端
### 🔒 隐私安全
- 💾 完全本地化部署，所有数据存在自己服务器
- 🚫 不需要上传数据到任何第三方服务器
- 🔐 不用担心账号信息、抽卡数据泄露
## 🚀 快速开始
### 1. 安装依赖
```bash
cd backend
pip install -r requirements.txt
```
如果不需要OCR功能，可以只安装基础依赖：
```bash
pip install fastapi uvicorn requests pydantic python-multipart
```
### 2. 启动服务
```bash
python main.py
```
### 3. 访问使用
打开浏览器访问：http://你的服务器IP:8777
就可以开始使用了！
## 📖 使用教程
### 第一步：添加账号
1. 点击「添加账号」按钮
2. 选择游戏类型，填写账号名称、服务器、UID
3. 确认添加
### 第二步：导入抽卡记录
#### 方式1：官方链接导入（推荐，最方便）
1. 打开游戏 → 抽卡页面 → 历史记录
2. 点击右上角「刷新」按钮，等待加载完成
3. 复制抽卡记录页面的链接
4. 点击对应账号的「导入数据」按钮，粘贴链接，点击「开始导入」
5. 系统会自动同步所有历史抽卡记录
#### 方式2：手动录入
适合补充少量遗漏的抽卡记录，直接填写信息添加即可
### 第三步：查看分析报告
点击「查看分析」按钮，就能看到完整的抽卡分析报告：
- 总抽数、五星数量、五星率
- 各个卡池当前水位和下次出货概率
- 历史五星出货记录
- 欧非评级和趣味评价

默认端口是8777，需要修改的话编辑 `main.py` 最后一行的port参数即可。
## 💡 使用小技巧
1. **定期同步**：建议抽完卡后定期同步数据，保证分析准确
2. **多设备访问**：手机和电脑在同一局域网下都能访问使用
3. **数据备份**：定期备份 `data/gachastats.db` 文件，防止数据丢失
4. **多账号对比**：可以同时添加多个账号，看看哪个号运气更好
